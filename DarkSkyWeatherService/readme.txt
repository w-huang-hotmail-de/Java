This is a simple console application which sources the DarkSky.net API to retrieve the wather information 
for the given localtion on the planet identified with latitude and longitude.

Please import this project into Eclipse IDE and run it.
Or register the related .jar files into CLASSPATH of Windows and run the command in console: java -cp bin com.statista.weather.app.App

The application is developed using Java SE 8 under Eclipse 2020 as a Java project (not Maven project).
The techniques used in this application are:
1. Java SE
2. unirest (http liberary)
3. darksky API
4. jackson
