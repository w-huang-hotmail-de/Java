This is a web-application using JFreeChart to show a pie Chart of global GDP (Gross domestic Product) 2019. 

Import this project into Eclipse IDE and run it on Tomcat Server.
Or register this project on a Tomcat Server and start this Tomcat server, then call the adress 
http://localhost:8080/JFreeChartDemo01/index.jsp on browser. 

This application is developed under Eclipse 2020 as a Dynamic Web Project.
The techniques used in this project:
1. Java EE
2. JFreeChart
3. Tomcat Server
